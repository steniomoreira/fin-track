# FinTrack 💰

**FinTrack** é uma aplicação moderna de controle financeiro pessoal desenvolvida com as tecnologias mais recentes do ecossistema web. O objetivo é ajudar usuários a gerenciarem suas transações, contas e metas de forma intuitiva e eficiente.

## 🚀 Tecnologias Utilizadas

- **Framework**: [Next.js 16](https://nextjs.org/)
- **Linguagem**: [TypeScript](https://www.typescriptlang.org/)
- **Estilização**: [Tailwind CSS](https://tailwindcss.com/)
- **Componentes**: [Shadcn UI](https://ui.shadcn.com/)
- **Banco de Dados**: [Prisma ORM](https://www.prisma.io/) com PostgreSQL
- **Autenticação**: [Better Auth](https://better-auth.com/)
- **Formulários**: [React Hook Form](https://react-hook-form.com/) & [Zod](https://zod.dev/)
- **Tabelas**: [TanStack Table](https://tanstack.com/table/v8)

## 🛠️ Como Iniciar

1. **Clone o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/fin-track.git
   ```

2. **Instale as dependências:**

   ```bash
   pnpm install
   ```

3. **Configure as variáveis de ambiente:**
   Copie o arquivo `.env.example` para `.env` e preencha com suas credenciais.

   ```bash
   cp .env.example .env
   ```

4. **Prepare o banco de dados:**

   ```bash
   pnpm prisma migrate dev
   ```

5. **Inicie o servidor de desenvolvimento:**
   ```bash
   pnpm dev
   ```
